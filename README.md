# 文件说明
## 1、`adualt.csv`
- 数据集来源：https://www.kaggle.com/datasets/wenruliu/adult-income-dataset/data
- 该数据集包含6个数字特征，8个离散字符特征，详细信息如下表：
 
|  特征  |  说明  |
|  ----  |  ----  |
|age | 年龄|
|workclass | 工作地点|
|fnlwgt | 一个州内数据集的一个观测代表的人数|
|education | 受教育程度|
|education-num | 受教育年数|
|marital-status | 婚姻状态|
|occupation | 职业|
|relationship | 关系|
|race | 种族|
|sex | 性别|
|capital-gain | 投资收益|
|capital-loss | 投资损失|
|hours-per-week | 每周工作时间|
|native-country | 国家|

| 分类种类 | 说明|
|  ---- |  ----  |
| >50K  |年收入大于50K $|
| <=50K |年收入小于等于50K $|

更多细节请查看：https://www.cs.toronto.edu/~delve/data/adult/adultDetail.html
